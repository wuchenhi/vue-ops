# vue 
金融订单支付系统前端界面

通过配置代理，解决跨域问题
登录、显示商品、创建订单、支付、查询订单支付状态完成支付等功能的实现与路由

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
